Variables

variables are used to store data that can be used and manipulated in a program. Variables act as containers for values like numbers, strings, objects, or functions

There are three ways to declare a variable:
  1. var (old way, avoid using it)
  2. let (modern and recommended for variables that can change)
  3. const (for variables that should not be reassigned)

1. Primitive Data Types (Stored by value)

  1. Number : Numeric values
  2. String : Text values
  3. Boolean : True or false
  4. Undefined : A variable declared but not assigned a value
  5. Null : Represents an intentional empty value

2. Non-Primitive (Reference) Data Types (Stored by reference)

  1. Object : Collection of key-value pairs
  2. Array : Ordered list of values
